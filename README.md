#一些设计想法
	原本要将每个页面都做成自适应的，即PC端和移动端运用媒体查询写到一个页面，实现PC端电子书化，移动端现代化。
	由于PC端实际操作时运用了iframe加载页面，为了开发便捷和切换方便，将每一个栏目的页面都写了两套。
	PC端在相应文件夹根目录下，移动端则在该文件夹的m目录下。
	两种页面由相应页面的JS来监控屏幕大小从而进行调节切换。
